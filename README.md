# DavidSan — Personal AI Operating System

DavidSan is a personal AI system built to help one person do their best work.

It is not a generic chatbot. It is designed around a specific person's context, goals, and working style.

## What problem it solves

Most people have strong experience but struggle to present it clearly. Context is scattered across notes, emails, and conversations. Generic AI tools give generic answers.

DavidSan works differently. It learns your context and helps you:

- present your professional value clearly and specifically
- decide what to work on and in what order
- draft outreach, summaries, and positioning that sounds like you
- stay focused on the opportunities that actually matter

## How it works

It runs a fine-tuned AI model locally on your machine. It asks you direct questions when it needs more context before giving a recommendation. It keeps a priority-driven workspace instead of a simple chat window.

Everything stays private by design. No data is sent to external services.

## Why it matters

This project explores what a truly personal AI assistant could look like — one that knows you, protects your data, and helps you operate at a higher level.

## Tech used

Python · Fine-tuned Phi-4 · Local LLM · Dashboard UI · Workflow automation
